El presente proyecto se considera una beta del proyecto final el cual tiene como principal objetivo  la realización 
de un software destinado a la gestión de reserva de espacios comunes para un condominio, el cual
tiene como propósito ayudar en el proceso de control de reserva, este nuevo sistema
busca sustituir el sistema manual que se emplea en la actualidad para una mayor comodidad
tanto para los administrativos como para los residentes. 

El sistema a crear es un sistema de plataforma Web, con la principal funcionalidad de llevar a cabo
la gestión reserva de espacios comunes en un condominio. Los residentes podrán escoger el espacio que
necesiten por un tiempo determinado para hacer la reserva y luego poder hacer uso del espacio, siempre y
cuando el espacio se encuentre disponible en ese horario para ser utilizado.
